## What I Built (Optional Extension)

As part of exploring the tools, I built:

**Growth Deconstruction Engine**

A simple web app that applies a 6-layer framework to analyze how any system grows:

1. Entry Point
2. Hook
3. Conversion Trigger
4. Retention Mechanism
5. Expansion Loop
6. Unfair Advantage


## Key Insight

Growth is context-dependent.
Instead of scoring systems, the framework identifies the **unfair advantage** that made growth work in a specific situation.
